

# CLI-ToDO

A simple command-line task manager application written in Go with colorful terminal output.

## Description

This application allows users to manage tasks through the command line. Each task is saved in a separate file, making it easy to organize and edit task lists.

## Features

- `add` - Adds a new task to the list.
- `show` - Displays the contents of task files.
- `done` - Marks a task as completed.
- `del` - Deletes a task.
- `help` - Shows brief information about available commands.
- `ext` - Exits the program.

## Installation

1. Make sure Go (version 1.XX or later) is installed.
2. Clone the repository.
3. Install dependencies:

```bash
go get github.com/1428Stef/CLI-ToDo-GO
```

3. Run the program:

```bash
CLI-ToDO.exe
```

Tasks are saved as separate files, which simplifies task management and storage.
